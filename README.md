## 📌 About

**PythonDailyGrind** is a curated collection of Python solutions to popular coding problems — primarily from LeetCode and similar platforms. Each file is a clean, well-commented Python script focused on problem-solving logic, algorithm design, and Pythonic coding practices.

Whether you're prepping for technical interviews or just grinding daily problems, this repo has you covered.

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/bharathkumar-24/PythonDailyGrind.git
   cd PythonDailyGrind
   ```

2. **Run any solution**
   ```bash
   python "3Sum.py"
   python "Contains Duplicate.py"
   ```

3. **Read, understand, modify** — each file is self-contained with problem context in comments.
